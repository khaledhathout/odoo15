Authenticator

Odoo Github Authenticator helps to authenticate github user to odoo. Thus helping github user to login on odoo with Github User ID. When user login with Github then It is registered with Odoo user along with its Github credentials.

With the help of the Odoo Github Oauth provider, a user can log in to Odoo through his Github id, and do the same work on Odoo as on Github.

Configuring Oauth Provider


  • Before using the module you need to configure Oauth provider by adding Client ID and Client Secret in Oauth provider configurations. To configure Oauth Provider Run debug mode, Go to Odoo Settings > Oauth Providers

  • After opening Github Oauth Provider add client ID and client secret.


  • If you want that Github Authorised user should login
    to odoo as portal user then select User Type : Portal User in Github Oauth provider section.

  • If you want logged in user should be internal then select User Type : Internal User.

After configuring the Oauth Provider when Github User login with Github.
  • If User is logging in for the very first time then User will be asked to enter github username and password.

  • Enter username and password of github and authorise the third party app to access github for logging in to odoo.

  • Once you are done logging in with Github Authorise the third party app so that It is granted to be accessed in Odoo. After successful authorisation you will be signed in as Github User in Odoo.
  • If admin has set User type in Oauth Provider as Internal User than you will be signed as Odoo Backend User.
  • If set Portal User as User Type then User will be logged in as Portal.
