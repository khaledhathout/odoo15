Organization Chart | OrgChart | Org Chart | Employee Hierarchy | Employee Chart

This module allows the hr users to quickly view the Employee Hierarchy using the existing Employee master, also the HR Manager will get the options for drag and drop, create , edit and delete options from the chart itself.

Installation

- Copy organization_chart module to addons folder

- Install the module normally like other modules

Features

- quickly create,edit and delete the employee hierarchy.

- drag and drop nodes

- print, reload, zoom in , zoom out and expand options

- export to png and pdf options

Settings -> Users-> User -> Human Resources -> Employees
